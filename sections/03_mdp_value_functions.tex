\section{Three Big Ideas}

\begin{frame}{Three Big Ideas in Offline Reinforcement Learning}
  Complementary approaches to improve behavior cloning:

  \vspace{0.35cm}
  \begin{enumerate}
    \item \textbf{Reweight the logged data} \\
    \emph{Advantage-Weighted Regression (AWR):} copy successful decisions more strongly.

    \vspace{0.25cm}
    \item \textbf{Learn conservative value estimates} \\
    \emph{Conservative Q-Learning (CQL):} be pessimistic where evidence is weak.

    \vspace{0.25cm}
    \item \textbf{Estimate values only from supported actions} \\
    \emph{Implicit Q-Learning (IQL):} avoid maximizing over unsupported actions altogether.
  \end{enumerate}

  \note{This is the roadmap for the second half. Emphasize that the talk is now organized around ideas, not papers or algorithms.}
\end{frame}

\begin{frame}{Generic Actor-Critic Offline RL}
  Most modern actor-critic Offline RL methods separate two steps:

  \vspace{0.15cm}
  \[
  \begin{aligned}
    &\text{estimate } Q,V \\
    &\Updownarrow \\
    &\text{improve policy}
  \end{aligned}
  \]

  \vspace{0.25cm}
  The three ideas intervene at different points:
  \begin{itemize}
    \item \textbf{AWR} mainly changes the improve-policy step.
    \item \textbf{CQL} and \textbf{IQL} focus primarily on value estimation.
  \end{itemize}

  \vspace{0.25cm}
  Policy updates need values; methods differ in how those values are made reliable.
  \note{This is the conceptual anchor for the second half, but avoid implying that every method uses weighted behavior cloning. AWR is a policy-improvement method. CQL and IQL are primarily value-estimation philosophies.}
\end{frame}

\section{Idea 1: Reweight the Logged Data}

\begin{frame}{Problem: Not All Demonstrations Are Equally Useful}
  Behavior cloning treats every logged decision equally.

  \vspace{0.4cm}
  But logged data often contains a mixture:
  \begin{itemize}
    \item strong operator decisions,
    \item reasonable routine decisions,
    \item choices that led to delay, rehandling, or poor downstream outcomes.
  \end{itemize}

  \vspace{0.35cm}
  \begin{block}{Question}
    Can we learn more from the better demonstrations?
  \end{block}
  \note{This starts Chapter 1. Keep it as a reminder, not a full recap of BC.}
\end{frame}

\begin{frame}{AWR: Weighted Behavior Cloning}
  Behavior cloning solves:
  \[
    \min_\theta \sum_i \ell\!\left(\pi_\theta(s_i),a_i\right)
  \]

  \vspace{0.15cm}
  AWR changes exactly one thing:
  \[
    \min_\theta
    \sum_i
    \textcolor{red}{w(s_i,a_i)}
    \,\ell\!\left(\pi_\theta(s_i),a_i\right),
    \qquad
    \textcolor{red}{w(s,a)=\exp\!\left(\beta A(s,a)\right)}.
  \]

  \vspace{0.25cm}
  \begin{block}{Intuition}
    Everything else is still supervised learning; high-advantage examples simply count more.
  \end{block}
  \note{This is the key equation for AWR. The red term is the only structural change from behavior cloning.}
\end{frame}

\begin{frame}{Representative Method: AWR}
  Advantage-Weighted Regression assigns a weight to each logged decision.

  \vspace{0.3cm}
  \begin{itemize}
    \item estimate long-term value,
    \item compare the logged action to typical behavior in the same state,
    \item assign a larger weight to positive-advantage decisions,
    \item train with weighted supervised learning.
  \end{itemize}

  \vspace{0.3cm}
  \begin{block}{What to remember}
    Good demonstrations count more.
  \end{block}

  \vspace{0.1cm}
  Representative algorithm: Advantage-Weighted Regression (AWR).
  \note{Do not re-derive the AWR equations; they were already introduced. This slide only labels the first big idea.}
\end{frame}

\begin{frame}{Remaining Problem: Where Do the Weights Come From?}
  AWR assumes reliable value estimates.

  \vspace{0.25cm}
  Its weights depend on advantage:
  \[
    A(s,a)=Q(s,a)-V(s).
  \]

  \vspace{0.15cm}
  If $Q$ or $V$ is unreliable, the weights become unreliable too.

  \vspace{0.35cm}
  \begin{block}{Next question}
    How should we estimate values offline?
  \end{block}
  \note{This transition leads directly to conservative value learning.}
\end{frame}
