\begin{frame}{Diffusion Policies Represent Richer Action Distributions}

  \begin{tikzpicture}[node distance=0.75cm,>=Latex]

    \node[draw,rounded corners,fill=Panel,right=of step2,minimum width=2.2cm] (action) {action};
    \draw[->,thick] (noise) -- (step1);
    \draw[->,thick] (step1) -- (step2);
    \draw[->,thick] (step2) -- (action);
  \end{tikzpicture}

  \vspace{0.55cm}
  \Large
  A diffusion policy can sample several plausible actions for the same state.

  \vspace{0.25cm}
  \normalsize
  IDQL combines IQL-style value learning with diffusion-based policy extraction.
  \note{Keep this short. The endpoint is that richer actors can represent multimodal action choices while the values still guide improvement.}
\end{frame}
